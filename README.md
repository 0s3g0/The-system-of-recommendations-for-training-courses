Задание по практики в университете
ТЗ:
Система рекомендаций учебных курсов
Цель:
Разработать прототип рекомендательной системы, предлагающей студентам элективные курсы на основе их оценок и текстового описания курсов.
Описание:
Команда работает с синтетическими или реальными данными о студентах (оценки по пройденным предметам) и описаниями курсов (текст). Требуется построить векторные представления курсов (например, на основе TF-IDF или bag-of-words) и для заданного студента найти наиболее похожие курсы, используя косинусное расстояние.
Что нужно реализовать:
•	загрузку данных об оценках студентов и описаниях курсов;
•	предобработку текстов (удаление стоп-слов, лемматизация/стемминг) и векторизацию с помощью numpy;
•	расчёт косинусной меры сходства между курсами или между профилем студента и курсами;
•	ранжирование курсов по убыванию сходства и вывод топ-5 рекомендаций;
•	проверку работы системы на нескольких примерах студентов;
•	визуализацию результатов (например, тепловая карта сходства или таблица с рейтингом).
Результат:
Программа на Python с модулями для обработки текста и расчёта сходства, а также отчёт с примерами рекомендаций и обоснованием выбора метрик.
